Detailed Description

Class to iterate on tuples of a relation implementation.

Warning:
The tuple iteration on relations should be used with care. Very large relations can result in large amount of run time.

The implementation of this iterator makes it advance on the relation after the call to the val(). It is not in any way an standard iterator.
