Detailed Description

Constant view on a relation variable

A constant view \(C\) for a ground relation \(G\) provides operations such that \(C\) behaves like the relation view with domain \([G,G]\).

  • To create a full view of arity n: 
       GRelation ur = create_full(n);
   ConstView u(ur);
  • To create an empty view of arity n: 
       GRelation er(n);
   ConstView e(er);
