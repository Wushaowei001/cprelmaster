Detailed Description

template<typename View>
class MPG::CPRel::ComplementView< View >

Complement relation view

A complement view \(C\) for a relation view \(R\) provides operations such that \(C\) behaves like the complement of \(R\).

Out of bound access.

Returns a relation (copy) that is known to not be part of the set of relations represented by the variable. \(oob = O \setminus lub \), wher \( O \) is the full relation.
