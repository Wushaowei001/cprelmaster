int  a

Creates a relation from a raw Bdd b and arity a.

Warning:
The constructor that allows the creation from a raw bdd is private for a good reason: it does not perform any check on the bdd or the given arity. The idea behind this method is to bypass this restriction but it should be used carefully.

RelationImpl timesU ( int  n,
bool  left 
) const

Computes the cross product of this with \( \mathcal{U}_n \).

As the cross product operation is not conmutative, the left parameter indicates whether to perform \( \mathcal{U}_n \times \mathrm{this}\) if left is true or \( \mathrm{this} \times \mathcal{U}_n\) otherwise.
