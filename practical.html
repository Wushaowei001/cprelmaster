Practical constraint programming on relations

Ground relation specification

There are several ways to specify a ground relation. The most common one is by specifying the set of tuples that conform it. See the documentation of the Tuple class for more information about how to create tuples.

For instance, the following function declares th ground relation R that contains the tuples (1,2,3), (3,0,2) and (99,30,42) and then prints it on the standard output. 

#include <vector>
#include <cprel/tuple.hh>
#include <cprel/grelation.hh>

void grelDecl(void) {
  using namespace MPG::CPRel;
  using namespace std;

  vector<Tuple> x0;
  x0.reserve(3);
  x0.push_back(Tuple(1,2,3));
  x0.push_back(Tuple(3,0,2));
  x0.push_back(Tuple(99,30,42));

  GRelation r(create(x0));

  cout << "R: " << r << endl;
}
