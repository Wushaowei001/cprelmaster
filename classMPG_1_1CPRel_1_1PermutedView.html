Detailed Description

template<typename View>
class MPG::CPRel::PermutedView< View >

Permuted relation view

A permuted view \(Xp\) for a relation view \(R\) provides operations such that \(Xp\) behaves like the permutation of \(R\) defined by the permutation description \( desc \).
